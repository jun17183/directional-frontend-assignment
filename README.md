# Directional 프론트엔드 과제

Next.js 기반의 게시판 및 데이터 시각화 대시보드 애플리케이션

## 프로젝트 실행 방법

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 사용한 기술 스택

- **Next.js 16** (Pages Router) - 라우팅 및 SSR
- **TypeScript** - 타입 안정성
- **React Query** - 서버 상태 관리 및 캐싱
- **MUI** - UI 컴포넌트 라이브러리
- **Recharts** - 데이터 시각화
- **React Hook Form + Zod** - 폼 관리 및 검증
- **Axios** - HTTP 클라이언트
- **react-hot-toast** - 알림 메시지

## 주요 구현 기능 요약

### 게시판

#### CRUD 작업
- **생성**: 게시글 작성 페이지에서 제목, 내용, 카테고리, 태그 입력
- **조회**: 게시글 목록 및 상세 페이지에서 게시글 확인
- **수정**: 작성한 게시글 수정 (제목, 내용, 카테고리, 태그 변경 가능)
- **삭제**: 삭제 확인 다이얼로그를 통한 안전한 삭제

#### 페이지네이션
- **커서 기반 페이지네이션**: offset 대신 cursor를 사용해 대용량 데이터 효율 처리
- **양방향 네비게이션**: 이전/다음 페이지 이동 지원
- **페이지 상태 유지**: 필터 및 검색 조건 유지하며 페이지 이동

#### 검색 및 필터링
- **제목 검색**: 실시간 검색어 입력 및 Enter 키로 검색 실행
- **카테고리 필터**: 공지사항, 질문답변, 자유게시판 중 선택
- **검색 초기화**: 필터 및 검색 조건 한 번에 초기화

#### 정렬
- **정렬 기준**: 제목 또는 작성일 선택
- **정렬 순서**: 오름차순/내림차순 선택
- **정렬 상태 표시**: 현재 정렬 기준과 순서를 UI에 표시

#### 폼 검증
- **Zod 스키마 검증**: 타입 안전한 폼 검증
- **제목 제한**: 최소 1자, 최대 200자
- **내용 제한**: 최소 1자, 최대 10,000자
- **금칙어 검증**: 클라이언트 사이드에서 부적절한 단어 차단
- **실시간 에러 표시**: 입력 필드별 에러 메시지 즉시 표시

#### 태그 기능
- **태그 추가**: Enter 키 또는 입력으로 태그 추가 (최대 10개)
- **태그 삭제**: 개별 태그 삭제 가능
- **중복 방지**: 동일 태그 중복 추가 방지

#### UI/UX
- **로딩 상태**: 데이터 로딩 중 스피너 표시
- **에러 처리**: 에러 발생 시 사용자 친화적 메시지 및 재시도 버튼
- **빈 상태 처리**: 검색 결과 없을 때 안내 메시지 표시
- **Toast 알림**: 성공/실패 알림 메시지 표시

### 데이터 시각화

#### 커피 브랜드 인기도
- **바 차트**: 브랜드별 인기도를 막대 그래프로 시각화
- **도넛 차트**: 브랜드별 인기도를 도넛 형태의 원형 차트로 표시
- **색상 구분**: 각 브랜드별 고유 색상으로 구분
- **인터랙티브 툴팁**: 호버 시 상세 정보 표시

#### 주간 무드 트렌드
- **스택형 면적 차트**: 주간별 무드 트렌드를 누적 면적 차트로 표시
- **3가지 무드**: 행복, 피곤, 스트레스 비율을 각각 다른 색상으로 표시
- **백분율 표시**: Y축에 백분율(%) 단위로 표시
- **날짜 포맷팅**: 주간 날짜를 MM/DD 형식으로 표시

#### 팀별 커피 소비와 생산성
- **멀티라인 차트**: 여러 팀의 데이터를 동시에 표시
- **이중 Y축**: 
  - 왼쪽 Y축: 버그 수 (실선, 원형 마커)
  - 오른쪽 Y축: 생산성 점수 (점선, 사각형 마커)
- **팀별 색상 구분**: Frontend(인디고), Backend(에메랄드), AI(앰버) 등 팀별 고유 색상
- **커스텀 툴팁**: 호버 시 해당 포인트의 커피 잔수, 버그 수, 생산성 점수를 팀별로 그룹화하여 표시
- **마커 구분**: 원형 마커(버그 수)와 사각형 마커(생산성)로 시각적 구분

#### 공통 기능
- **반응형 디자인**: 화면 크기에 따라 자동 조절
- **로딩/에러 상태**: 각 차트별 로딩 및 에러 상태 처리
- **툴팁 z-index**: 툴팁이 최상단에 표시되도록 설정

### 인증
- **JWT 기반 인증**: 토큰 기반 인증 시스템
- **자동 토큰 첨부**: Axios 인터셉터로 모든 요청에 자동으로 토큰 첨부
- **401 에러 처리**: 인증 실패 시 자동으로 로그인 페이지로 리다이렉트
- **로그인 상태 유지**: localStorage에 토큰 저장하여 새로고침 시에도 로그인 상태 유지
- **보호된 라우트**: 인증되지 않은 사용자는 로그인 페이지로 자동 리다이렉트
