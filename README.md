# Directional 프론트엔드 과제

Next.js 기반의 게시판 및 데이터 시각화 대시보드 애플리케이션

## 프로젝트 실행 방법

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 사용한 기술 스택

- **Next.js 16** (Pages Router) - 라우팅 및 SSR
- **TypeScript** - 타입 안정성
- **React Query** - 서버 상태 관리 및 캐싱
- **MUI** - UI 컴포넌트 라이브러리
- **Recharts** - 데이터 시각화
- **React Hook Form + Zod** - 폼 관리 및 검증
- **Axios** - HTTP 클라이언트
- **react-hot-toast** - 알림 메시지

## 주요 구현 기능 요약

### 게시판
- **CRUD 작업**: 게시글 생성, 조회, 수정, 삭제
- **커서 기반 페이지네이션**: 대용량 데이터 효율 처리
- **검색 및 필터링**: 제목 검색, 카테고리 필터 (공지사항, 질문답변, 자유게시판)
- **정렬**: 제목/작성일 기준 오름차순/내림차순
- **금칙어 검증**: 클라이언트 사이드 검증으로 부적절한 내용 차단

### 데이터 시각화
- **커피 브랜드 인기도**: 바 차트 및 도넛 차트로 브랜드별 인기도 표시
- **주간 무드 트렌드**: 스택형 면적 차트로 행복/피곤/스트레스 비율 시각화
- **팀별 커피 소비와 생산성**: 멀티라인 차트 (이중 Y축)
  - 실선: 버그 수 (원형 마커)
  - 점선: 생산성 점수 (사각형 마커)
  - 팀별 색상 구분 및 커스텀 툴팁

### 인증
- JWT 기반 인증
- Axios 인터셉터로 자동 토큰 첨부 및 401 에러 처리
